## 一、搜索语法速查

### 1.1 基础搜索

| 语法 | 说明 | 示例 |
|-----|-----|-----|
| `关键词` | 全文搜索 | `error` |
| `A and B` | 与 | `error and timeout` |
| `A or B` | 或 | `error or warn` |
| `A not B` | 非 | `error not debug` |
| `(A or B)` | 括号 | `(error or warn) and mqtt` |

### 1.2 字段搜索

| 语法 | 说明 | 示例 |
|-----|-----|-----|
| `字段:值` | 精确匹配 | `level:error` |
| `字段:*值*` | 模糊匹配 | `message:*timeout*` |
| `字段:*` | 字段存在 | `errorKey:*` |
| `not 字段:值` | 字段不等于 | `not level:debug` |

### 1.3 通配符

| 语法 | 说明 | 示例 |
|-----|-----|-----|
| `前缀*` | 前缀匹配 | `mqtt*` |
| `*后缀` | 后缀匹配 | `*error` |
| `*中间*` | 包含匹配 | `*time*` |
| `?` | 单字符 | `wxid_????` |

### 1.4 短语和特殊字符

| 语法 | 说明 | 示例 |
|-----|-----|-----|
| `"短语"` | 短语搜索 | `"connection refused"` |
| `"[xxx]"` | 搜索方括号 | `"[error]"` |
| `"中文"` | 中文短语 | `"群总数"` |

### 1.5 数值

| 语法 | 说明 | 示例 |
|-----|-----|-----|
| `>` | 大于 | `status > 400` |
| `>=` | 大于等于 | `status >= 400` |
| `<` | 小于 | `status < 300` |
| `<=` | 小于等于 | `status <= 200` |
| `in [a b]` | 范围 | `status in [400 500]` |

## 二、SQL 语法速查

### 2.1 SELECT

```sql
SELECT *                          -- 所有字段
SELECT message, __time__          -- 指定字段
SELECT message AS msg             -- 别名
SELECT DISTINCT level             -- 去重
SELECT COUNT(*) AS cnt            -- 聚合
```

### 2.2 WHERE

```sql
WHERE level = 'error'             -- 等于
WHERE level != 'info'             -- 不等于
WHERE level IN ('error', 'warn')  -- 列表
WHERE level NOT IN ('debug')      -- 不在列表
WHERE __time__ > 1706000000       -- 比较
WHERE __time__ BETWEEN a AND b    -- 范围
WHERE message LIKE '%error%'      -- 模糊匹配
WHERE message IS NOT NULL         -- 非空
```

### 2.3 ORDER BY

```sql
ORDER BY __time__ ASC             -- 升序
ORDER BY __time__ DESC            -- 降序
ORDER BY level, __time__ DESC     -- 多字段
```

### 2.4 LIMIT 和 OFFSET

```sql
LIMIT 100                         -- 限制返回数
LIMIT 100 OFFSET 50               -- 分页
```

### 2.5 GROUP BY

```sql
GROUP BY level                    -- 单字段分组
GROUP BY level, __source__        -- 多字段分组
```

### 2.6 HAVING

```sql
HAVING COUNT(*) > 100             -- 过滤聚合结果
HAVING cnt > 100                  -- 使用别名
```

## 十一、常用查询模板

### 11.1 基础查询

```sql
-- 查看最新日志
* | SELECT * ORDER BY __time__ DESC LIMIT 100

-- 某用户日志
用户名 | SELECT * ORDER BY __time__ DESC LIMIT 100

-- 错误日志
level:error | SELECT * ORDER BY __time__ DESC LIMIT 50
```

### 11.2 统计查询

```sql
-- 日志级别分布
* | SELECT level, COUNT(*) AS cnt GROUP BY level ORDER BY cnt DESC

-- 错误率
* | SELECT ROUND(SUM(CASE WHEN level='error' THEN 1 ELSE 0 END)*100.0/COUNT(*), 2) AS error_rate

-- 按小时统计
* | SELECT DATE_FORMAT(FROM_UNIXTIME(__time__), '%Y-%m-%d %H:00') AS hour, COUNT(*) AS cnt
    GROUP BY DATE_FORMAT(FROM_UNIXTIME(__time__), '%Y-%m-%d %H:00') ORDER BY hour
```

### 11.3 排名查询

```sql
-- 错误 Top 10
level:error | SELECT SUBSTR(message, 1, 80) AS error, COUNT(*) AS cnt
              GROUP BY SUBSTR(message, 1, 80) ORDER BY cnt DESC LIMIT 10

-- 活跃用户 Top 10
* | SELECT REGEXP_EXTRACT(message, '\[([^\]]+)\]', 1) AS user, COUNT(*) AS cnt
    GROUP BY REGEXP_EXTRACT(message, '\[([^\]]+)\]', 1) ORDER BY cnt DESC LIMIT 10
```
