# Galaxy-Client 项目总览

> 高途微信助手 - Electron 桌面客户端

---

## 一、项目简介

### 1.1 项目定位

**Galaxy-Client**（高途微信助手）是一个基于 Electron 的 Windows 桌面客户端应用，用于实现微信和企业微信的自动化操作。主要服务于销售和客服场景，提供群发消息、自动回复、群管理等功能。

### 1.2 核心能力

| 能力分类 | 具体功能 |
|---------|---------|
| **消息发送** | 群发文本、图片、视频、文件、链接、小程序、名片 |
| **好友管理** | 通过好友申请、删除好友、修改备注、标签管理 |
| **群聊管理** | 创建群聊、修改群名、群公告、踢人、邀请入群、退出群聊 |
| **数据同步** | 同步好友列表、群聊列表、群成员信息 |
| **消息接收** | 接收并转发微信/企微消息到云端 |

### 1.3 支持平台

- **操作系统**: 仅支持 Windows（因逆向服务依赖）
- **微信版本**: 3.7.5.31、3.8.0.18
- **企业微信版本**: 4.1.2.6011、4.1.3.6008

## 五、开发与构建

### 5.1 环境要求

```bash
# Windows 系统要求
- Windows 10/11
- Python 3.10.x (node-gyp 需要)
- Visual Studio 2017/2019 + Windows SDK

# Node.js 版本
- Node.js 16.x 或 18.x (与 Electron 20 兼容)
```

### 5.2 开发步骤

```bash
# 1. 安装依赖
npm install

# 2. 下载逆向服务文件（需指定版本）
node ./scripts/downloadDevFile.js 5.4.2-release01

# 3. 启动开发模式
npm run start:dev
```

### 5.3 构建步骤

```bash
# 设置版本号
set Version=5.4.2-release01

# 执行构建
node ./scripts/build.js
```

## 八、快速定位

### 按功能定位

| 功能需求 | 查看文件 |
|---------|---------|
| 了解项目启动流程 | `src/electron.js` → `event/app.js` → `init/window.js` |
| 了解消息如何下发 | `core/mq/mqttClientBase.js` → `dispatch-center/dispatchInBound.js` |
| 了解如何与逆向通信 | `core/reverse/initIpcTask.js` → `dll/clibrary.js` |
| 了解登录流程 | `business/convert-service/loginService.js` |
| 了解群发消息 | `business/task-mqtt/mqttChatService.js` |
| 了解任务回执 | `dispatch-center/handle/msgHandleBase.js` |

### 按问题定位

| 遇到问题 | 查看位置 |
|---------|---------|
| MQTT 连接失败 | `core/mq/mqttClientBase.js` 的 `on('error')` |
| IPC 通信异常 | `core/reverse/initIpcTask.js` 的 `run()` |
| 任务执行失败 | 搜索日志 `[codeError]` 或 `level: 'error'` |
| 消息未送达 | 查看 `dispatch-center/handle/msgHandleBase.js` 三段式逻辑 |
