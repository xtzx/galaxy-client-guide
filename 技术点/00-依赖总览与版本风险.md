## 二、依赖分类详解

### 2.1 Electron 框架相关

| 依赖包 | 版本 | 功能说明 |
|--------|------|----------|
| `electron` | 20.0.2 | **桌面应用框架**。⚠️ 版本锁定，因为 ffi-napi 在新版本中不兼容 |
| `@electron/remote` | ^2.0.10 | 允许渲染进程调用主进程模块（Electron 14+ 需要单独安装） |
| `electron-better-ipc` | ^2.0.1 | 主进程与渲染进程之间的 IPC 通信封装 |
| `electron-log` | ^4.4.8 | **日志系统**。自动写入文件，支持多级别日志 |
| `electron-store` | ^8.1.0 | **本地持久化存储**。基于 JSON 文件 |
| `electron-updater` | ^6.1.1 | **自动更新**。支持增量更新 |
| `electron-builder` | ^23.6.0 | **应用构建打包**。生成 Windows 安装包 |

### 2.2 逆向通信层（最关键）

| 依赖包 | 版本 | 功能说明 |
|--------|------|----------|
| `ffi-napi` | ^4.0.3 | ⭐**核心依赖**。Node.js 调用 C/C++ DLL 的桥梁 |
| `ref-napi` | - | ffi-napi 的依赖，提供 C 语言数据类型在 Node.js 中的表示 |

**为什么 Electron 版本锁定在 20.0.2？**
- ffi-napi 依赖 Node.js 的 N-API
- Electron 20.0.2 使用的 Node.js ABI 版本与 ffi-napi 兼容
- 升级 Electron 会导致 ffi-napi 无法加载 DLL

## 三、版本风险提示

### 3.1 高风险依赖

| 依赖 | 风险等级 | 问题描述 | 建议 |
|------|---------|---------|------|
| `electron` 20.0.2 | 🔴 高 | 版本过旧（2022年8月），有安全漏洞 | 评估 ffi-napi 替代方案后升级 |
| `sequelize` 7.0.0-alpha.2 | 🔴 高 | Alpha 版本不稳定 | 降级到 6.x 稳定版 |

### 3.2 中等风险依赖

| 依赖 | 风险等级 | 问题描述 | 建议 |
|------|---------|---------|------|
| `webpack` 4.x | 🟡 中 | 版本过旧 | 升级到 5.x |
| `node-fetch` 2.x | 🟡 中 | 主版本过旧 | 考虑升级到 3.x（ESM） |

### 3.3 无用依赖

| 依赖 | 问题描述 | 建议 |
|------|---------|------|
| `crypto` | 空包，Node.js 内置 | 移除，直接使用内置模块 |
| `fs` | 空包，Node.js 内置 | 移除，直接使用内置模块 |

## 四、依赖安装特殊说明

### 4.1 原生模块编译

由于 `ffi-napi` 和 `sqlite3` 包含原生代码，安装时需要编译环境：

```bash
# Windows 环境要求
1. Python 3.10.x（不要用 3.11+）
2. Visual Studio 2017/2019 + Windows SDK
3. node-gyp 全局安装

# 安装步骤
npm install --global node-gyp
npm install --global windows-build-tools  # 可选

# 安装项目依赖
npm install
```

### 4.2 Electron Rebuild

打包前需要重新编译原生模块：

```bash
# 重新编译
npx electron-rebuild

# 或在 package.json 中配置
{
  "scripts": {
    "postinstall": "electron-rebuild"
  }
}
```

## 五、依赖更新策略

### 5.1 可安全更新

- `electron-log`
- `electron-store`
- `axios`
- `uuid`
- `fs-extra`
- `async-lock`

### 5.2 需谨慎更新

- `mqtt` - 主版本更新可能有 API 变化
- `sharp` - 原生模块，需要重新编译
- `sequelize` - ORM 更新可能影响数据库操作

### 5.3 暂不更新

- `electron` - 受 ffi-napi 限制
- `ffi-napi` - 核心依赖，更新风险高
- `sqlite3` - 原生模块，版本稳定优先
