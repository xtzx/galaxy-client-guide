## 配置文件位置

```
galaxy-client/
├── config/
│   ├── weixinzhushou/
│   │   ├── build.yml        ← 本系列分析的配置文件
│   │   └── runtime.yml      ← 运行时配置（打包时复制为 app-config.yml）
│   ├── damai/
│   │   ├── build.yml
│   │   └── runtime.yml
│   └── ...                  ← 其他产品线
└── package.json             ← 构建命令定义
```

构建命令示例：

```bash
electron-builder -w -c config/weixinzhushou/build.yml -p never
```

- `-w`：构建 Windows 平台
- `-c`：指定配置文件路径
- `-p never`：不发布（仅本地构建）
