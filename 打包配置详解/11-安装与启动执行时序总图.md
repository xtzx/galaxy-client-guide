## 三、卸载过程时序

```
用户通过「控制面板 → 程序和功能 → 高途微信助手 → 卸载」
│
▼
┌─────────────────────────────────────────────────────────────────────────┐
│ 卸载流程                                                                │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  ① UAC 权限提升                                                        │
│                                                                         │
│  ② NSIS 卸载程序执行                                                    │
│     a. 停止正在运行的进程                                               │
│     b. 删除安装目录下的所有文件                                         │
│     c. 删除桌面快捷方式                                                 │
│     d. 删除开始菜单快捷方式                                             │
│     e. 清理注册表卸载信息                                               │
│                                                                         │
│  ③ 保留用户数据（deleteAppDataOnUninstall: false）                      │
│     %APPDATA%\weixinzhushou\ 目录不删除                                │
│     → 用户重装后数据可恢复（但前面安装时已删除 db 文件）               │
│                                                                         │
│  ④ hosts 文件中添加的微信更新屏蔽条目不会被清理                        │
│     → 需要用户手动删除 127.0.0.1 dldir1.qq.com                        │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## 四、覆盖安装（升级）时序

```
用户双击新版本安装包
│
▼
┌─────────────────────────────────────────────────────────────────────────┐
│ 覆盖安装特殊流程                                                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  与首次安装的区别：                                                      │
│                                                                         │
│  ① customInit 中的 suicideMark 生效                                    │
│     → 旧版主程序检测到标记 → 主动执行退出流程：                        │
│       a. 通过 MQTT 上报所有微信退出登录                                 │
│       b. 关闭 Java 服务                                                 │
│       c. 关闭 BasicService                                              │
│       d. 主程序退出                                                     │
│     → 4 秒后 taskkill 兜底强杀残留进程                                 │
│                                                                         │
│  ② $INSTDIR 已存在                                                      │
│     → prevent_wx_update.bat 可以正常执行                                │
│     → 安装向导默认显示旧版安装目录                                      │
│                                                                         │
│  ③ NSIS 覆盖安装                                                        │
│     → 直接覆盖 $INSTDIR 下的所有文件                                   │
│     → 新版 DLL、exe、源码替换旧版                                      │
│     → runtime.yml → app-config.yml 也被新版替换                        │
│                                                                         │
│  ④ 数据库被删除                                                         │
│     → data.db 和 sqlite.db 在 customInit 中被删除                      │
│     → 新版启动后会重新创建空数据库                                      │
│     → 用户需要重新同步好友列表、群信息等                                │
│                                                                         │
```

```
│                                                                         │

│     → autoUpdater.quitAndInstall()                                      │
│     → 关闭当前应用                                                      │
│     → 启动新版安装包                                                    │
```
